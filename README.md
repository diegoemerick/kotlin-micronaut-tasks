# Bepy Task

Service created for the study of micronaut and kotlin, also has a GRPC part

## How to run this project

1 - ```./gradlew clean```

2 - ```./gradlew build```

3 - ```./gradlew generateProto```


----
##### Warn: Before starts the service run the postgres through docker-compose

```docker-compose up -d```
